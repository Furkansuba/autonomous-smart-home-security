# Autonomous Smart Home Security
Capstone project repository for an autonomous smart home security system.
## Main Stack
- ESP32
- MQTT
- Node.js + Express
- MongoDB Atlas
- JWT + RBAC
- Kotlin Android App
- React Admin Panel
- Firebase Cloud Messaging
- AWS EC2
## Repository Structure
- contracts: interface contracts, payload schemas, API rules
- backend: Node.js/Express backend
- android: Kotlin Android resident app
- admin-web: React admin panel
- firmware: ESP32 firmware
- docs: project documentation
- scripts: helper scripts
- tests: integration and scenario tests
